YouTube Tutorial
================
If you prefer a video tutorial instead of written instructions, this available near the top of the https://github.com/jonathanfox5/gogadget/ webpage.

Written Tutorial
================

1. Download the latest version of "gogadget_windows.zip" from https://github.com/jonathanfox5/gogadget/releases/

2. Extract the zip wherever is convenient to you. Inside the folder, you will find "windows_installer.ps1". Double click it, wait for it to complete and you are done!

3. You can then run the tool by typing the following command into command prompt or powershell:
gogadget

4. [Optional] You can install all of the models required for your chosen language. Type the following to get the instructions:
gogadget install
