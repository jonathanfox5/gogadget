YouTube Tutorial
================
If you prefer a video tutorial instead of written instructions, this available near the top of the https://github.com/jonathanfox5/gogadget/ webpage.

Written Tutorial
================

1. Download the latest version of "gogadget_windows_installer_vxxx.exe" from https://github.com/jonathanfox5/gogadget/releases/

2. Run the installer with the default settings and you are done!

3. You can then run the tool by opening it from the start menu. You can type the following for instructions.
gogadget

4. [Optional] You can install all of the models required for your chosen language. Type the following to get the instructions:
gogadget install
